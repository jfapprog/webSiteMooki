Marcas HTML utilizadas nesta página (por ordem de utilização):

  • Usei a marca <!DOCTYPE html> para identificar a versão do documento: HTML5.
  • Usei as marcas <!-- --> para inserir um comentário com o nome e número de aluno.
  • Usei as marcas <html> e </html> para definir a raíz do documento.
  • Usei as marcas <head> e </head> para definir o cabeçalho do documento.
  • Usei as marcas <title> e </title> para definir o título do documento, apresentado na barra de título da janela do browser.
  • Usei a marca <meta> para definir metadados do documento, nomeadamente a codificação de caracteres: UTF-8.
  • Usei a marca<link> para realizar a ligação para a folha de estilo: minhaFolhaEstilo.CSS.
  • Usei as marcas <body> e </body> para definir a zona do conteúdo do documento apresentado pelo browser.
  • Usei as marcas <header> e </header> para definir a zona de cabeçalho do conteúdo do documento.
  • Usei as marcas <p> e </p> para definir o parágrafo 'Mokki'.
  • Usei as marcas <b> e </b> para destacar a bold a palavra 'Matemática'.
  • Usei as marcas <div> e </div> para definir uma divisão para o conteúdo.
  • Usei as marcas <a> e </a> para definir uma hiperligação para a página ano09.html.
  • Usei as marcas <img> para inserir a imagem com a hiperligação para a página ano09.html.
  • Usei as marcas <h5> e </h5> para definir o título 'Recebe novidades "Mooki"' com o nível de heading 5.
  • Usei as marcas <form> e </form> para inserir o formulário para indicação do nome e email para receção das novidades.
  • Usei as marcas <footer> e </footer> para definir a zona de rodapé do conteúdo do documento.
  • Usei a marca <br> para inserir uma quebra de linha entre os elementos de entrada do formulário.
  • Usei as marcas <input> e </input> para inserir elementos de entrada no formulário, nome e email.
  • Usei as marcas <code> e </code> para inserir o código referente às tags referidas nesta secção.
  • Usei as marcas <i> e </i> para destacar a itálico, nesta secção, a palavra 'browser'.

    • p (nas regras "header p:first-child" e "header p:last-child")

Propriedades formatadas nos diversos seletores (por ordem alfabética):

  • Usei background-color para definir a cor de fundo de diferentes elementos.
  • Usei border para definir as diferentes propriedades das linhas delimitadoras de diferentes elementos.
  • Usei border-collapse para colapsar numa única linha as linhas delimitadoras das células das tabelas.
  • Usei border-bottom para definir as diferentes propriedades da linha delimitadora inferior de um elemento do tipo "input".
  • Usei border-left para definir as diferentes propriedades da linha delimitadora esquerda dos elementos da classe "bloco".
  • Usei border-left para definir as diferentes propriedades da linha delimitadora esquerda dos elementos da classe "bloco".
  • Usei bottom para definir o posicionamento absoluto do elemento da classe "menu".
  • Usei box-shadow para definir um efeito de sombra dos elementos da classe "bloco" e do tipo "input".
  • Usei color para especificar a cor do texto de cada elemento.
  • Usei cursor para especificar o tipo de cursor quando um elemento do tipo "input[text=submit]" é apontado.
  • Usei display para definir o tipo de caixa utilizada.
  • Usei float para especificar que alguns elementos da classe "bloco" devem flutuar à esquerda ou à direita
  • Usei font-size para especificar o tamanho de fonte.
  • Usei font-weight para definir o texto dos títulos a mais grosso.
  • Usei height para definir a altura dos elementos do tipo "img".
  • Usei line-height para definir a altura da linha de texto.
  • Usei list-style-type para definir o tipo de marcacdor das listas não ordenadas.
  • Usei margin para definir as quatro margens de diferentes elementos.
  • Usei margin-left para definir a margem esquerda de alguns elementos.
  • Usei margin-top para definir a margem superior de alguns elementos.
  • Usei opacity para especificar o grau de opacidade do elemento da classe "menu".
  • Usei overflow para definir que, nos elementos da classe "footerBloco", o conteúdo é escondido quando ultrapassar os limites da caixa.
  • Usei padding para especificar o espaço da margem interna de diferentes elementos.
  • Usei padding-bottom para especificar o espaço da margem interna inferior de alguns elementos.
  • Usei position para especificar o positionamento fixo no parte inferior do elemento da classe "menu"
  • Usei text-align para definir como central o alinhamento horizontal do texto nas células das tabelas.
  • Usei text-decoration para que o texto das hiperligações não apareça sublinhado.
  • Usei text-indent para definir a identação da primeira linha dos subtítulos.
  • Usei text-shadow para aplicar sombra aos títulos e subtítulos.
  • Usei width para definir a largura de diferentes elementos.
  • Usei z-index para definir a ordem pela qual o menu aparece (à frente do texto).
