Lógica e teoria dos conjuntos

Proposições e valores lógicos

Definição

Designa-se por proposição toda a expressão susceptível de ser "verdadeira" ou "falsa".

Exemplo

Indica se as seguintes expressões são proposições:

  1. -52 =52
  2. Lisboa é uma cidade bonita.
  3. -3<-π
  4. É agradável estudar.

Sugestão de resolução

  1. Proposição.
  2. Não é proposição (depende da opinião).
  3. Proposição.
  4. Não é proposição (depende da opinião).

Exercício

Indica se as seguintes expressões são proposições:

  1. Todos os múltiplos de 4 são múltiplos de 2.
  2. -23
  3. Um retângulo é um paralelogramo.
  4. Amanhã é domingo.

Resolução

Nota: Responde "S" ou "N".

Definição

Os atributos "verdadeira" ou "falsa" são designados por valores lógicos.

Exemplo

Indica o valor lógico das seguintes proposições:

  1. -52 =52
  2. Um losango é um quadrado.
  3. -3<-π
  4. Um triângulo isósceles pode ser retângulo.

Sugestão de resolução

  1. Verdadeira.
    -52=25 ; 52=25
  2. Falsa.
    Um quadrado é um losango.
  3. Falsa.
    -3>-π
  4. Verdadeira.
    Os catetos de um triêngulo retângulo podem ser iguais.

Exercício

Indica o valor lógico das seguintes proposições:

  1. -43 =43
  2. 56 é múltiplo de 4.
  3. O coeficiente da função f, cuja expressão analítica é f(x)=2x-4 , é 4.
  4. A soma dos ângulos internos de um pentágono é 560 graus.

Resolução

Nota: Responde "V" ou "F".
